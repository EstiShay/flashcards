JavaScript (JS)

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more. The + operator, and the += operator can also be used to concatenate (add) strings.

Variables

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Variables Naming Conventions

Use lowerCamelCase when naming JavaScript variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word.

Functions

A function is a block of code that performs an action and returns a result; optionally takes arguments.

Methods

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

What you pass into a function or method. Some methods take one or more arguments that provide the method with additional information to help it perform its action.

Parameters

A variable that's assigned to the argument.

Return

Once called, many functions will provide a result of some kind. This is known as returning, and the result is often referred to as a return value.

Chaining methods

Method chaining is a technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively. For example, "prompt("Type something in lowercase:").toUpperCase();" will create an upper case version of the return of this prompt.

Strings

Data type that represents the exact text of whatever is enclosed in the quotes.

Booleans

A data type: true or false.

Undefined

An object (such as a variable) without a defined value.

NaN

NaN stands for 'not a number.'' NaN is a type of number

Escape

The \\ around either side of the text above is called an escape. It tells JavaScript that the " that comes right after it is not the end of the string, but just a character inside the string.

Alert

The alert() method displays an alert box with a specified message and an OK button. An alert box is often used if you want to make sure information comes through to the user.

Comments

Anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

jQuery

jQuery is a concise and fast JavaScript library that can be used to simplify event handling, HTML document traversing, and much more!

Attributes

Additional information provided to a tag. For instance, the href attribute in an tag provides the URL a link should travel to.
